The modal element can be used to show notices or secondary content above the z-level of the page.

Basic usage

<x-modal> shows by default. Use the HTML5 attribute hidden to hide it.

Overlay is transparent layer that sits over the main content, but below the modal. Use the attribute overlay to enable it.ÃŸ

The attribute overlay-tap-hide allows users to tap on the overlay to close the modal.

   <h1>HI</h1>
</x-modal>
